Input Validator

Input validator is a tiny library without dependencies that is used to validate data from single input.

Instalation

  • Install with Bower: bower install input-validator
  • Downlod from GitHub

Link library before closing body tag

<script src="js/input-validator.min.js"></script>

Link css styles to the head of document (optional)

<link rel="stylesheet" href="css/input-validator.min.css">

Usage

Recomended HTML structure

<div class="input-group">
  <label for="email">Your Email:</label>
  <input type="email" class="iv-input" id="email" name="email">
</div>

Javascript Validation

// Select input from DOM tree
var input = document.querySelector('input');

// Pass it in validate function
var validation = validate(input)
  .clear()
  .required()
  .max(120)
  .match(email);

// If validation passed do something
if (validation.isValid()) {
	...
}
