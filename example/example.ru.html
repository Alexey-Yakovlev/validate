Input Validator

Input validator - небольшая библиотека без зависимостей, предназначенная для валидации данных, полученных из одного тега input.

Документация Примеры Смотреть на Github

Утсановка

  • Установите с помощью Bower: bower install input-validator или скачайте zip архив на GitHub
  • Подключите библиотеку перед закрывающим тегом body
    • <script src="js/input-validator.min.js"></script>
  • Подключите css файлы (необязательно)
    • <link rel="stylesheet" href="css/input-validator.min.css">
  • Или воспользуйтесь готовым шаблоном
    • <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Input Validation</title>
  <link rel="stylesheet" href="css/input-validator.min.css">
  <link rel="stylesheet" href="css/styles.css">
</head>
<body>
	
  <script src="js/input-validator.min.js"></script>
  <script src="js/scripts.js"></script>
</body>
</html>
